## ‚ö° INSTALASI DAN SETUP

### Clone Repository
```git clone https://github.com/RyyzDev/sistemkasir-laravel.git```
```cd sistemkasir-laravel```

### Install Vendor
```composer install```

### Install NodeJS
```npm install```
```npm run dev```

### Buat File ENV
```cp .env.example .env```

### Generate Key
```php artisan key:generate```

### Buat Seed Migrate
```php artisan migrate --seed```

### Jalankan server:
```php artisan serve```
